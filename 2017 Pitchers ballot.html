1 Plot of all pitchers IP/ERA by ballot positionÂ¶

All players with pitching statistics are classified into a ballot position which can be one of LHP, RHP.

Pitcher handedness was determined by the CBN CIC web page.

There are a small number of players with pitching stats whose handedness is not shown on the CBN CIC web page (Those players are shown in red in the plot below). Since the pitching stats for these players did not qualify for ballot selection no effort was made to classify as LHP or RHP.

2 Pitching ballot positionsÂ¶

Below there is a section for each ballot position, which shows a plot of all players assigned to the ballot position, followed by a table with the selected players and their season stats.

Basic selection criteria: IP >= 60 and ERA <= 5.0

Each plot has two blue division lines that represent the IP and ERA thresholds for that position. Only players in the LOWER RIGHT area will be selected.

We can adjust the IP and ERA threshold for each position so we can get the desired number of players on the ballot for that position.
